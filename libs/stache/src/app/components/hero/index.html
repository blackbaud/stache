The stache-hero tag is used to add a Hero with a background image, title, and subheading information to your Stache page.
  • overlayOpacity — Decimal value for how much your image should be darkened. Defaults to 0.4, acceptable values are any decimal between 0 and 1.
  • backgroundImageUrl — Set this to the name of your image, prepended with ~/assets​/. eg: ~/assets​/foo.jpg. All images and other static assets should be stored in the src/assets directory.
